#  ЖОСКИ ТАСК

## Описание
API для управления задачами на FastAPI с поддержкой PostgreSQL и in-memory хранилища (потому что я сначала сделал просто на постгре, а потом до меня дошло что база то у меня локально и теперь тут два варианта, но если вдруг будет желанием можно свою базочку присоеденить и всё запрацюе. UPD: я больше не делал ничего ин мемари, но и старое не убрал, вот такая делюга).

## Возможности
- Создание задач
- Получение всех задач
- Получение конкретной задачи
- Обновление задачи
- Удаление задачи

## Технологический стек
- **FastAPI** — веб-фреймворк
- **SQLAlchemy** — ORM для работы с БД
- **PostgreSQL** — основная база данных
- **Pydantic** — валидация данных
- **Uvicorn** — ASGI сервер



### Запуск приложения
```bash
uv run uvicorn main:app --reload
```

## Режимы работы
- **PostgreSQL режим** — если база данных доступна
- **In-Memory режим** — автоматический fallback если нет доступа к БД

## Валидация
- Заголовок задачи: 1-255 символов
- Описание: максимум 5000 символов

## Новый функционал
- Аутентификация пользователей через JWT:
	- POST /auth/register — регистрация пользователя (email + пароль)
	- POST /auth/login — вход и получение access token (JWT)
- Паттерн Repository для работы с `User` (`repositories/user_repository.py`)
- Энедпоинты задач защищены — доступ по Bearer токену (JWT). Только владелец может управлять своими задачами.
- Хранение пользователей в БД (SQLAlchemy `User` модель). Пароли хешируются (bcrypt via `passlib`).
